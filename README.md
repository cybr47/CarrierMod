# CarrierMod

A simple, lightweight LSPosed module to customize carrier name and add lockscreen charging stats.

## Features
* **Custom Carrier Name:** Change or completely hide mobile network name. Works system-wide (Lockscreen, Quick Settings, and Settings app).
* **Live Charging Stats:** Replaces default lockscreen charging text with real-time stats: `Battery % • Current (A) • Voltage (V) • Wattage (W)`.

## Requirements
* Rooted Android device (Magisk, KernelSU, etc.)
* LSPosed Framework installed
* Android 12 to 16

## How to Use
1. Download and install the APK from the **Releases** tab.
2. Enable **CarrierMod** in LSPosed Manager.
3. Tap the **Gear Icon** (⚙️) on the CarrierMod page inside LSPosed to open the settings.
4. Make your changes, hit Save, and restart SystemUI.
